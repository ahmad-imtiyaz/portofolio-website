# **Portfolio Website – Ahmad Imtiyaz Najih**

## **Project Overview**

Portfolio Website ini merupakan situs web personal yang saya buat untuk menampilkan profil, keahlian, proyek-proyek, dan kontak profesional saya sebagai pengembang web. Situs ini dirancang dan dibangun dengan teknologi web modern agar responsif, efisien, dan mudah diperluas sesuai kebutuhan. ([GitHub][1])

Tujuan utama dari project ini adalah memberikan **wadah profesional online** yang:

* Memperlihatkan pengalaman dan kemampuan pengembangan web.
* Memberikan akses cepat ke proyek portofolio.
* Menyediakan kontak profesional yang mudah dijangkau.

## **Features**

Fitur utama yang ada di portfolio ini antara lain:

* Halaman **Beranda** berisi ringkasan profil singkat.
* Bagian **Tentang Saya** untuk deskripsi latar belakang dan keahlian.
* Tampilan **Proyek Portofolio** yang terstruktur.
* Form atau link kontak profesional.
* Navigasi responsif dan antarmuka yang mudah digunakan. ([GitHub][1])

## **Tech Stack**

Portfolio Website ini dibangun menggunakan teknologi berikut:

* **HTML5** — markup struktur konten utama.
* **CSS3 & TailwindCSS** — styling dan tata letak responsif.
* **ReactJs** — interaksi dinamis pada UI.
* **Vite** — build tooling modern untuk performa optimal.
* **Netlify** — deployment dan hosting (opsional).
* **Git & GitHub** — version control dan kolaborasi. ([GitHub][1])

## **Installation / Setup**

Ikuti langkah di bawah untuk menjalankan project secara lokal:

1. Clone repository:

   ```bash
   git clone https://github.com/ahmad-imtiyaz/portofolio-website.git
   ```
2. Masuk ke direktori project:

   ```bash
   cd portofolio-website
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Jalankan server development:

   ```bash
   npm run dev
   ```
5. Buka browser dan akses:

   ```
   http://localhost:3000
   ```

> Pastikan kamu sudah menginstal **Node.js** minimal versi terkini.

## **Usage**

Setelah berjalan secara lokal, kamu dapat:

* Mengubah konten di berkas `index.html` atau komponen lain.
* Menambah proyek baru di bagian portofolio.
* Menyesuaikan styling di file CSS atau Tailwind config.
* Deploy ke platform hosting seperti GitHub Pages atau Netlify.

## **Folder Structure**

Direktori utama project tersusun sebagai berikut:

```
/
├── public/                 # Aset statis seperti gambar, favicon
├── src/                    # Sumber kode utama
|   ├── assets/             # Gambar dan media lain
|   ├── styles/             # Styling Tailwind / CSS
|   ├── components/         # Komponen UI
|   └── main.js             # Entry point aplikasi
├── index.html              # Halaman utama
├── package.json            # Konfigurasi dependensi proyek
├── tailwind.config.js      # Konfigurasi TailwindCSS
└── vite.config.js          # Konfigurasi build tool Vite
```
